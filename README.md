# Student Depression Analysis
## 📊 Tools Used
- PostgreSQL — data cleaning & transformation
- Tableau Public — interactive dashboard

## 🔗 Live Dashboard
[Click here to view the dashboard](https://public.tableau.com/app/profile/namrata.chaurasia/viz/StudentDepressionAnalysis_17806649485420/StuentCountAnalysis)

## 📁 Dataset
- 503 student records
- 11 features: sleep duration, academic pressure, 
  financial stress, family history, depression status etc.
- Added columns: age_group, depression_flag, id

## 🔍 SQL Steps Performed
1. Imported raw CSV into PostgreSQL
2. Standardized Gender column
3. Added Age Group column (18-21, 22-25, 26-30, 30+)
4. Added Depression Flag (0/1)
5. Added Index column
6. Analyzed column distributions
